Darmowa, legalna baza wiedzy + lekki silnik liczenia. Jeden klik i masz zestaw startowy: pobieranie otwartych danych, indeks tekstowy, dashboard oraz „artefakty” do pracy badawczej.

Co jest w środku

Open Base
fetcher z checksumami, manifest otwartych datasetów, opcjonalny FTS5, workflow GitHub Actions (tylko meta).
Vault
destylaty (MDR), protokół ekspansji, backlog badań, wizualizacja ζ(1/2+it).
Dashboard
KPI: zadania, błędy narzędzi, self‑check; gotowy do rozbudowy o p95/ECE.

Jak uruchomić (lokalnie)

unzip open_base_bootstrap.zip -d open-base && cd open-base
python -m venv .venv && source .venv/bin/activate   # Windows: .venv\Scriptsctivate
pip install -r requirements.txt
python fetch.py --config config.yaml --out ./data

Zachowujemy licencje i ToS. Nie pobieramy „ciężarów” na CI; zaczynamy od API/małych zrzutów.

Publikacja (za darmo)

Najłatwiej wrzucić publicznie na GitHub i włączyć „Pages”. Pliki statyczne z tej strony działają od ręki.
